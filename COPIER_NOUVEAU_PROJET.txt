# Fichiers à copier pour démarrer un nouveau projet

## MÉTHODE 1: Utiliser GitHub Template (Recommandé)

1. Aller sur https://github.com/gbottazzoli/research-project-template
2. Cliquer "Use this template" (bouton vert)
3. Créer votre nouveau repo
4. Cloner: git clone votre-nouveau-repo
5. cd nouveau-repo && cat START_HERE.md

## MÉTHODE 2: Clone Direct

```bash
git clone https://github.com/gbottazzoli/research-project-template.git mon-projet
cd mon-projet
rm -rf .git
git init
cat START_HERE.md
```

## UTILISATION

Une fois les fichiers copiés:

1. cat START_HERE.md
2. cat QUICK_PROMPT.md
3. Copier TOUT le prompt dans Claude Code
4. Remplir les informations sur votre projet
5. Claude crée automatiquement:
   - Structure de dossiers
   - Documentation
   - Configuration
   - Agents configurés
6. Commencer à travailler immédiatement!
